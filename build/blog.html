시작하기

Sapcy Playbook 은 티스토리 등에서 옮겨 온 기술 글과 도구 모음을 한곳에 모아 둔 문서 허브입니다.

이 사이트에서 할 수 있는 것​

기능설명
왼쪽 사이드바주제(클라우드, 쿠버네티스, CS, 백엔드)별로 글 목록을 탐색합니다.
오른쪽 목차긴 문서에서는 h2~h4 제목이 목차로 표시되어 섹션으로 바로 이동할 수 있습니다.
태그각 글 하단 또는 태그 모아보기에서 주제(예: AWS, Java, 알고리즘)별로 글을 모아 봅니다.
검색상단 검색창으로 문서 전체를 검색합니다.

문서 구성​

왼쪽 메뉴와 동일한 분류입니다.

구분내용
클라우드 → AWSEKS, Argo CD, IAM·클러스터 연동 등 AWS·운영 관련 글
쿠버네티스동작 원리·Pod·스케줄러·컨트롤러·containerd/CRI 등 개념 노트, KYAML·kubectl/인증서 글
CSHTTP 캐시 헤더 등 기초 CS·네트워크
백엔드 → 코딩알고리즘·문제 풀이(PCCP, 백준 등)
백엔드 → Java문법, JVM, JIT 등 Java 런타임
백엔드 → SpringSpring 프레임워크·IoC 문서

다른 페이지​

상단 메뉴에서 바로 갈 수 있는 페이지입니다.

  • 프로젝트 - WebSeal, Sealed Secret 등 실험용 도구 UI
  • 정보 - 소개·이력 등

팁​

  • 코드 블록은 언어별로 구문 강조가 적용됩니다.
  • 다크 모드는 우측 상단(또는 하단) 테마 전환으로 바꿀 수 있습니다.

궁금한 점이나 오류는 GitHub 저장소 이슈로 알려 주세요.
